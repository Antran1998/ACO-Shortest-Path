\section{Thesis Structure}

The report is organised as follows:
\begin{itemize}
  \item Chapter 1: Introduction, motivation, goals and scope.
  \item Chapter 2: Overview and Approach, high-level description of the implemented ACO and the set of proposed improvements.
  \item Chapter 3: Detailed Improvements, dataset/benchmark description and per-improvement: idea, implementation, benefits, and experimental results.
  \item Chapter 4: Conclusion, summary and future work.
\end{itemize}
